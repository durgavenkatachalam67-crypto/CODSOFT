## ✅ Tasks Completed

### Task 1 — Titanic Survival Prediction
Predict whether a passenger survived the Titanic disaster based on features like age, gender, ticket class, and fare.
- **Type:** Binary Classification
- **Models:** Logistic Regression, Random Forest, SVM, KNN
- **Dataset:** Titanic-Dataset.csv

---

### Task 2 — Movie Rating Prediction
Predict the rating of an Indian movie based on features like genre, director, and actors using regression techniques.
- **Type:** Regression
- **Models:** Linear Regression, Ridge, Random Forest, Gradient Boosting
- **Dataset:** IMDb Movies India.csv

---

### Task 3 — Iris Flower Classification
Classify Iris flowers into three species (Setosa, Versicolor, Virginica) based on sepal and petal measurements.
- **Type:** Multi-class Classification
- **Models:** Logistic Regression, Decision Tree, Random Forest, SVM, KNN
- **Dataset:** IRIS.csv

---

### Task 4 — Sales Prediction Using Python
Forecast product sales based on advertising expenditure across TV, Radio, and Newspaper platforms using regression.
- **Type:** Regression
- **Models:** Linear Regression, Ridge, Lasso, Random Forest, Gradient Boosting
- **Dataset:** advertising.csv

---

### Task 5 — Credit Card Fraud Detection
Identify fraudulent credit card transactions using classification algorithms with handling of severe class imbalance.
- **Type:** Binary Classification
- **Models:** Logistic Regression, Random Forest
- **Techniques:** SMOTE oversampling, Precision, Recall, F1-Score evaluation
- **Dataset:** creditcard.csv
